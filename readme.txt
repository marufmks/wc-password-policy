Enhance your WooCommerce store's security with customizable password policies and requirements.

== Description ==

WooCommerce Password Policy allows store owners to implement robust password requirements for their customers. Ensure account security by enforcing password complexity rules and managing password lifecycles.

= Features =

* Set minimum password length
* Require uppercase letters
* Require lowercase letters
* Require numbers
* Require special characters
* Password expiration (90 days)
* Password reuse prevention
* Customizable error messages
* Real-time password validation

= Requirements =

* WordPress 5.6 or higher
* WooCommerce 3.0 or higher
* PHP 7.4 or higher

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/wc-password-policy` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to WooCommerce > Settings > Password Policy to configure the plugin settings.

== Frequently Asked Questions ==

= Does this work with WooCommerce? =

Yes, this plugin requires WooCommerce to be installed and activated.

= Can I customize the error messages? =

Yes, you can customize all error messages from the Password Policy settings page in WooCommerce settings.

= Does this affect existing user passwords? =

No, the policy only applies to new passwords when users register or change their passwords.
